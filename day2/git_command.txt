git status - проверить статус, если есть красный файлы, значит что-то не добавлено или удаленно
git add - добавить какой-либо файл или папку если в ней есть файлы, если она пустая ничего добавить не сможешь
git commit -am "здесь пишешь твои изменения" - сommit сохраняет твои изменеия
git push -u origin master - отправляешь это все ни github
